

# Lead Management CRM

A modern Lead Management CRM built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

* User Login & Logout
* Protected Dashboard
* Add New Leads
* Edit Existing Leads
* Delete Leads
* Search & Filter Leads
* Lead Status Analytics
* Responsive Modern UI
* Dark Mode Support
* MongoDB Database Integration

## Tech Stack

### Frontend

* React.js
* Vite
* Axios
* React Icons
* Chart.js

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Installation

### Clone Repository

```bash
git clone https://github.com/vaibhav378-vi/lead-management-crm.git
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create `.env` file inside backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
